Project: WhatsApp Bulk Messenger



Project Description:



A lightweight **Flask web app** for sending **personalized WhatsApp messages** to a list of contacts via browser automation. Users **upload an Excel file containing name and phone columns** and provide a message template with a {name} placeholder; the app then **drives Microsoft Edge through Selenium** to open WhatsApp Web, log in via QR code, and send each contact a personalized message in sequence.



Prerequisites:



* Python 3.9 or later
* Microsoft Edge Browser
* Excel Sheet (.xlsx file) with 'name' \& 'phone' columns
